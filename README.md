# 🚀 Coder’s Nest

Coder’s Nest is an AI-powered collaborative coding IDE designed for real-time team development, intelligent coding assistance, and scalable workspace management.

It combines a powerful in-browser IDE with AI-driven development tools and real-time collaboration features, enabling developers to build software together seamlessly.

---

## ✨ Features

### 🧠 AI-Powered Development
- Local AI integration for offline code assistance
- Cloud AI APIs for advanced code generation & debugging
- Context-aware suggestions inside editor

### 👨‍💻 Real-Time Collaboration
- Live multi-user coding environment
- Simultaneous editing (Google Docs style)
- Real-time cursor tracking & presence system

### 📁 Project Management
- Team-based workspace creation
- Role-based access control (Owner, Admin, Member)
- Project-level organization system

### 🔐 Authentication System
- Secure user registration & login
- Email verification (OTP-based system)
- Password hashing with bcrypt
- Session-based authentication design

### ⚙️ Backend Architecture
- FastAPI-based modular backend
- Scalable service-oriented structure
- Database integration with SQLAlchemy ORM

---

## 🏗 Tech Stack

- **Frontend:** React + Monaco Editor
- **Backend:** FastAPI (Python)
- **Database:** MySQL
- **Real-time:** WebSockets
- **AI Layer:** Local Models + External APIs
