# Coder’s Nest

Coder’s Nest is an AI-powered collaborative coding IDE for real-time team development, AI-assisted programming, and workspace management.

It provides a unified environment where developers can code together, use AI for assistance, and manage projects efficiently.

---

## Features

AI-Powered Development
- Local AI integration for offline code assistance
- Cloud AI APIs for code generation and debugging

Real-Time Collaboration
- Multi-user live coding environment
- Simultaneous editing with real-time sync

Project Management
- Team workspaces
- Role-based access control

Authentication
- Secure login and registration
- OTP-based email verification
- Password hashing and session design

---

## Tech Stack

Frontend: React + Monaco Editor  
Backend: FastAPI  
Database: MySQL  
Real-time: WebSockets  
AI: Local + Cloud APIs
